

1)  Undo the Changes in Local Area ?
	
		git checkout -- filename.py

		(or)

		git checkout -- .




2)  Undo the Changes in Staging Area ?

		git restore --staged filename.py



3)  Undo the Changes in Working (or) Local Repository ?

		git revert 123456789 

		(or)

		git revert HEAD



4)  Reset 
		

		git reset --hard 123456789

		(or)

		git reset 123456789

		(or)

		git reset HEAD
